oracle用户被锁

oracle的账户多次输入错误密码会被锁定(十次)，尤其是在新建立的时候动不动被锁 记录一下账户被锁的解锁步骤 请完整复制代码内容

# 1.打开cmd,输入

sqlplus/noloog
1

应该会显示时间，这时候表明成功

# 2.以dba身份登录

conn / as sysdba
1

这样是不需要输入用户名密码就可以登录的

# 3.输入解锁语句

alter user system account unlock;
1

不输入 ; 会不执行 不知道为啥 有了解的烦请留言解答
