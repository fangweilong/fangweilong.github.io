2023年度总结

# 2023年度总结

2023一晃神，又过去了。好像还没过足滋味，就来到了24年。那就趁现在，给自己做一波年终总结。

# 锻炼

# 跑步

温度上来的时候重拾了跑步，晚上动一动跑一跑，一天基本也就3-4公里的量。十月份咳嗽了一个多月，又荒废了。

# 爬山

今年兴致上来了，小山倒是爬过好多次，夜爬也有过三次。
第一次夜爬，几个人开着导航在惠山脚下绕了好几圈楞是找不对路，爬完到家都已经3点多。
十二月份还一个人夜爬，仍然也有不少人在爬山。夜爬人少，不会出现一早上大量的人群拥挤的现象，强烈安利一下

# 习惯

2023年最大的一个习惯改变可能就是删掉了“阅读”APP，戒掉了沉迷小说的日子。对于手机的需求量骤降，也释放出来了更多零碎的碎片化时间。

# 购物

# 家居

# 灯具

对于智能家居布置，有一个很重要的环节，就是灯具。
PDD、TB、JD都能找到很多的智能灯具，可以连接小爱、天猫、小度等，但是使用下来设计层次不齐，而且也不符合这么多年养成的使用习惯。且开关留在门口的位置，临时进出手动开关更方便快捷。后来设计洗手间人来灯亮、人走灯灭的时候就没考虑直接换灯具，而是购买了几个智能开关面板（凌动开关）。
实际使用体验上来说，凌动开关确实比直接更换灯具更省时间，低成本就能保留使用习惯，但是没办法调节灯具色温、亮度。（闺女现在已经会自己喊小爱同学开关电视看宝贝玖玖、开关灯）

# 联动

米家的产品在做家居联动上确实有优势，各种三方厂家的产品补充了米家平台，小黄鱼上也有很多个人的魔改米家产品。我在小黄鱼上淘了一个魔改版“人在传感器”，放置在卫生间的墙壁上，联动卫生间灯和走道灯，实现了人在客厅就能知道卫生间有没有人，而不用敲门拧把手。现在连闺女都知道卧室找不到人的时候，走道灯亮着就说明在洗手间。

# 1688

以前一直都听过1688的大名，但是一直没用过。最近在逛酷安的时候，有不少人写了在1688找货的教程贴，去尝试了一下，确实找到货源地会比PDD都便宜，织物含量更清晰。PDD很多店铺问织物含量永远只会告诉你有什么东西，但是成分都是一个谜。但是1688不少店铺是不包邮的，需要提前看清楚，或者是到多少件包邮。 30%的羊毛袜是暖和的，下次再试试更高含量。
