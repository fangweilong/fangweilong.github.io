Elastic_Search-添加split自定义分词方法

# 版本

5.4.2

# 准备

需要删除索引，否则会报已存在的错误

# 正文

  • 以 ; 分割，如果需要其他符号的自己修改 pattern 后面的内容
  • fielddata是为了aggs，不需要的可以不要
PUT 索引名
{
  "settings": {
    "analysis": {
      "analyzer": {
        "自定义的分词器名": {
          "type": "pattern",
          "pattern":"自定义的分词字符，相当于 split(';')中的 ; "
        }
      }
    }
  },
  "mappings": {
    "doc": {
      "properties": {
         "字段名": {
          "type": "text",
          "analyzer": "上面定义的分词器名",
          "search_analyzer": "上面定义的分词器名",
          "fielddata": true
        }
       }
    }
  }
}
