jquery 节流防抖

# 前言

# 在一些大数量或者有延迟性的操作下，为了页面不要每次输入或者点击的时候闪烁，我们需要对接口进行调用频率限制。

直接上代码

# 代码

//需要定义在方法外
let timeout;
function throttling(){
	//先清理
	clearTimeout(timeout)	
	timeout = setTimeout(() => {
		//在此处写调用的方法，可以实现仅最后一次操作生效
    	console.log('abc')
	}, 3000)
}
