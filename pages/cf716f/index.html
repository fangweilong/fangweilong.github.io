# 刷机步骤

# 好久没刷机

之前用的 乌堆小透明 (opens new window) 的刷机包，很好用更新还快，卡刷线刷一体，直接安装magisk等。

但是从2023年年初开始，包就不更新了，MIUI本身的bug也在修复，那就自己刷吧。

# 需要知道的一些常识

  • 进入rec模式: 关机状态下，长按 电源键和音量上键
  • 进入fastboot模式：关机状态下，长按 电源键和音量下键

# 刷机过程

# 卡刷（已经有rec）

1. 进入rec模式
2. 双清/三清
3. 选择刷机包直接安装即可

# 线刷

我本来想刷个开发版，结果刷的版本一直有问题。先是备份的数据无法读取，然后又是不能挂载sdcard,最后进入系统一直没法打开app。一气之下直接清掉刷稳定版

  • 从xiaomirom站点下载最新的稳定版的rom包
  • 线刷。这里有各种各样的工具，搞机助手、奇兔等各种各样的，不展开记录。

# 刷入magisk

参考文档 (opens new window)

  • 安装magisk delta的apk
  • 解压上面下载的rom包,找到一个 boot.img ,放到手机的磁盘上
  • 使用magisk修补，选中 boot.img ,等待一会就会输出一个 'magisk_patched-随机字符串.img' 的一个镜像文件，复制到电脑上
  • [搞机助手]上打开adb工具，输入命令。注意自己的.img文件的路径
    fastboot flash boot ./magisk_patched_随机字符串.img
1
  • 重新启动系统
  • 打开magisk，会提示需要修补环境并重启，确定即可。

# 刷入twrp

  • 使用[搞机助手-引导模式-输入REC]功能，选择从rec网站下载的适配机型的rec文件，直接刷入即可。有些机型可能需要选择 A/B分区 刷入
